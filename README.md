# Human-in-the-Loop Fake News Detection System

## Overview
This project detects Fake vs Real news using Machine Learning (TF-IDF + Logistic Regression) with Human-in-the-Loop (HITL) decision support.

## Features
- Fake vs Real classification
- Confidence-based prediction
- Human review escalation
- Streamlit frontend
- Visualization of model performance

## Model
- TF-IDF Vectorization
- Logistic Regression classifier
- Confidence threshold: 95%

## Workflow
1. Input news article
2. Model predicts class
3. Confidence is checked
4. Low confidence → human review
5. Final decision displayed

## Tech Stack
Python, Scikit-learn, Pandas, Streamlit, Matplotlib
